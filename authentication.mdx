---
title: Authentication
description: "How to authenticate with the Stablepay API"
---

All Stablepay API endpoints require a Bearer token issued by Stablepay.

## Using your API key

Pass the key in the `Authorization` header on every request:

```http
Authorization: Bearer spay_live_xxxxxxxxxxxxxxxxxxxx
```

Contact [hello@stablepay.io](mailto:hello@stablepay.io) to get an API key for your account.
