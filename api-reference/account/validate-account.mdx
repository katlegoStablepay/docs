<Note>
  This endpoint is coming soon. Account validation will allow you to verify recipient details before creating a payment, reducing failed transfers.
</Note>

## What this will do

Before creating a payment, you'll be able to submit the recipient's account details and receive confirmation that the account exists and can receive funds. This is particularly useful for:

- **Mobile money**: Verify a phone number is registered with the specified network.
- **Bank accounts**: Confirm account number and bank code are valid before sending.

## Planned request

```json
{
  "accountNumber": "0712345678",
  "accountType": "MOMO",
  "currency": "KES"
}
```

## Planned response

```json
{
  "valid": true,
  "accountName": "John Doe",
  "accountNumber": "0712345678"
}
```
